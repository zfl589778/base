项目：面向高校的大学生创业平台
业务方向：
	面向高校的大学生创业平台，本项目旨在帮助高校孵化更多互联网创业项目或高新企业，促进学生就业，发掘学生创新能力和商业潜能，通过学校本体与学生本体线上互动及促进线下交流的方式，
	为学生具体的创业思路提供解决方案和理论指导，从而实现学生的人生价值和理想，同时政府可用于相关于学生创业的宣传，达到政府、高校、学生三赢的效果
市场可行性分析：
市场分析：
	唐山互联网人才匮乏，缺少高新企业，因此政府鼓励互联网创业，在政策上给予创业者很多帮助，包括设立高新区，减免租税等。本项目顺从政府意愿，市场上目前缺少具有类似针对性的相关产品，具有一定的市场可行性
适用人群：
	政府：用于宣传高新企业创业进展，鼓励学生创业，促进学生就业
	高校：为学生提供创业解决方案，包括但不限于提供导师、理论指导、实践操作等帮助
	学生：上传自己的创业思路和相关报告，由平台审核，通过后学生可以接受学校的针对性帮助，平台为学生和学校提供包括但不限于线上互动、线下交流等互动方式
技术可行性分析：
技术方案v1：
	性质：基于B/S结构的javaweb项目
	应用场景：互联网
	应用用户：具有一定针对性的普通用户
	系统架构：使用网上的第三方开源框架（基础架构：ssm,redis,mysql,dubbo）
	开发方式：敏捷开发，必要环节采用外包兼职方式，例如页面设计等，将产生费用算入成本
	达到效果：v1包含主要功能，完整流程，和谐美观的页面风格，用户指导等demo，用于进行下一步推广和拉动投资
人力配置v1：
	用户界面设计，包括设计、切图、js效果、简单编码、调整浏览器兼容性：1-2人，经验：从事过类似需求网站的UI设计工作，要求作品交互性强，和谐美观，提供方便的使用接口
	
名词解释：
	B/S结构：即浏览器-服务器结构，交互流为用户->浏览器->服务器，例如平时浏览的网页，对应的为C/S结构，交互流为用户->桌面客户端->服务器
	javaweb项目：即使用java语言作为服务器开发语言的网站项目
	v1：第一版，仅实现基础功能
	demo：样例，不可用于生产使用，仅用作展示
	ssm,redis,mysql,dubbo：软件开发专业术语
